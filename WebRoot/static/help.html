说明

目的
  • 吃货QQ群：299780910 欢迎大家加群交流/指导/帮忙完善这个网站
  • 提供西安工程大学临潼校区学校食堂及后门的餐馆的各种饭菜的详细信息.
  • 网站的访问者可以浏览、点评、打分饭菜/餐馆的质量。
  • 自己写着玩的，绝对不是以盈利为目的（其实也没有可行的盈利方案～.～）
环境
  • jsp+servlet+javabean+mysql
  • bae+linux
操作说明
  • 浏览信息：这不用说明了吧
  • 添加餐馆/饭菜信息：不需要登陆，直接填写信息提交就可以
  • 添加图片：也不需要登陆，直接选择图片，上传
  • 添加评论：需要登陆，然后对相应饭菜信息添加评论
关于登陆/注册
  • 仅仅评论时需要登陆账号，简单注册一下就OK了
  • 登陆安全稍微处理了一下，但是不保证绝对安全
  • （ps：我在数据库端对用户密码做了md5散列加密，这也许是最奇葩的处理加密的方式了Orz）
  • 后期如果这个网站活跃了，我可以加入百度的账号认证API，这样就可以直接用百度账号登陆了
关于饭菜图片
  • 餐馆/饭菜的信息需要各位同学的帮忙添加完善
  • 饭菜的图片尽量做到真实拍摄，吃饭的时候可以顺手拍一张
  • 图片文件保存在web文件目录里面，如果后期图片信息比较多的话，可能会用百度的BCS API
其他
